Frequently Asked Questions

Everything you need to know about the Full Sail Code Challenge.

What is the Full Sail Code Challenge?

The Full Sail Code Challenge is an online competitive programming competition where students solve coding and algorithmic problems against the clock. Participants compete individually by writing working, efficient code to solve increasingly challenging programming problems.

The event helps students strengthen the technical problem-solving skills used in software engineering, game development, artificial intelligence, technical interviews, and programming competitions.

Who can participate?

Students who are currently enrolled in Full Sail's programming-focused degree programs.

When is the competition?

June 26, 2026 at 6:00 PM – June 27, 2026 at 11:59 PM (ET)

The competition takes place during a two-day online event window.

Is this an online event?

Yes. This is so the competition can be available to online and campus students.

What platform will be used?

HackerRank.

Do I compete alone or on a team?

This is an individual competition.

What kinds of problems should I expect?

Competitors should expect a variety of programming challenges including some of the following:

  • Algorithms
  • Data Structures
  • Recursion
  • Searching & Sorting
  • Mathematics
  • String Processing
  • Optimization

Problems are inspired by classic competitive programming and technical interview exercises. The exact challenge set will remain confidential until the competition begins.

Is this like ACM or ICPC?

It is inspired by ACM/ICPC-style competitive programming, but this is an individual competition rather than a team event.

What programming languages can I use?

Python, JavaScript, C++ are fully supported languages for these challenges. C++ is expected to be fully supported.

How is scoring determined?

Each problem is worth a different number of points based on its difficulty.

Does efficiency matter?

Yes. Efficient solutions may perform better on larger or more demanding test cases. Efficiency may also be used as tie breaker if needed.

Can I use AI tools?

To help prepare you for collegiate coding competitions and technical assessments, this competition will be hosted on HackerRank using its built-in proctoring system. During the competition, you will use a single monitor and grant HackerRank access to your screen and webcam to ensure a fair and authentic competition experience. Unless otherwise stated in the official rules, AI tools are not permitted during the competition. These guidelines reflect the standards commonly used in collegiate programming competitions and help you gain experience in a professional competitive environment.

Will there be prizes?

Yes! Nintendo Switch 2s are planned for 1st, 2nd, and 3rd place.

Will there be a leaderboard?

Yes. HackerRank will provide a live contest leaderboard throughout the event.

Do I need to install anything?

Generally, no. You'll need:

  • A modern web browser
  • A reliable internet connection
  • A HackerRank account
  • Webcam
  • Your preferred IDE (if coding locally)
Will there be support?

Faculty may assist with technical issues but cannot provide programming hints or solutions.

How should I prepare?
  • Practice algorithms and data structures
  • Review recursion
  • Understand Big-O analysis
  • Strengthen debugging skills
  • Solve problems under time pressure
I've never done competitive programming before. Should I still participate?

Absolutely! The event is designed for students of all experience levels.

Do I need to solve every problem?

No. Most participants will not finish every problem. Success comes from solving as many as you can within the competition time.
